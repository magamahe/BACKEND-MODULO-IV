# Backend Avanzado: Integración de Bases de Datos, Sequelize, Firebase y Seguridad en APIs

## 📚 Descripción del Curso

Este curso se centra en la **integración y gestión de bases de datos** dentro de aplicaciones backend modernas.

Durante la cursada se trabaja con:

- Bases de datos **SQL**
- Bases de datos **NoSQL**
- **ORM Sequelize**
- Integración con **MongoDB**
- Seguridad en **APIs backend**

Además se abordan **buenas prácticas de desarrollo**, diseño de bases de datos y conceptos fundamentales de **ciberseguridad aplicados al backend**.

---

# 🎯 Objetivo del curso

Aprender a:

- Integrar bases de datos en aplicaciones backend
- Utilizar **Sequelize** como ORM
- Administrar bases de datos **SQL y NoSQL**
- Conectar **Node.js con MongoDB**
- Implementar **seguridad en APIs**
- Prevenir vulnerabilidades comunes en aplicaciones web
